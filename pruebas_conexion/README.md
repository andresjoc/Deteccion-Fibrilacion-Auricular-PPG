# Carpeta de Pruebas de Conexión con Dispositivos PPG

Este directorio agrupa todas las pruebas realizadas para la **conexión y adquisición de señales PPG (fotopletismográficas)** de **relojes inteligentes**, tanto de **marca comercial** como **genéricos**.

---

## Propósito

El objetivo de esta carpeta es **centralizar los experimentos y desarrollos iniciales** relacionados con la **comunicación, captura y validación de señales PPG** provenientes de diferentes tipos de dispositivos.

Aquí se documentan:

- Los **procedimientos de conexión** a través de SDKs, APIs o protocolos BLE.
- Las **pruebas de adquisición de datos** desde cada tipo de reloj.
- Los **resultados y observaciones** sobre la calidad, frecuencia y estabilidad de las señales obtenidas.
