# Carpeta de Dispositivos Genéricos

En este directorio se registran los intentos, pruebas y resultados relacionados con la **obtención de señales PPG (fotopletismográficas)** a partir de **dispositivos genéricos o no comerciales**, es decir, relojes o sensores sin SDK oficial o sin marca reconocida.

---

## Propósito

Documentar los procesos de **conexión, adquisición y análisis preliminar** de señales PPG provenientes de relojes o sensores que:

- No cuentan con un **SDK o API oficial**.
- Son de **marca desconocida o genérica**.

Estos ensayos buscan determinar la **viabilidad técnica** de usar estos dispositivos en entornos experimentales o educativos, así como evaluar **las señales PPG** obtenidas frente a las de dispositivos de marca.
