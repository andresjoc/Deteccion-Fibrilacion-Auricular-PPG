# Carpeta de Dispositivos de Marca

En este directorio se documentan los avances, pruebas y resultados obtenidos durante la **adquisición de señales PPG (fotopletismográficas)** a partir de **relojes inteligentes de marca comercial**, adquiridos por la universidad para el desarrollo del proyecto.

---

## Propósito

Registrar todo el trabajo relacionado con la **obtención y manejo de señales PPG desde dispositivos de marca**, utilizando los **SDKs, APIs o herramientas oficiales** provistas por cada fabricante.

Los datos y scripts aquí almacenados servirán para:

- Evaluar las señales PPG que cada marca ofrece.
- Desarrollar procesos de extracción, sincronización y almacenamiento de datos.
