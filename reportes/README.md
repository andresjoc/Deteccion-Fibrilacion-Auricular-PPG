# Carpeta de Reportes del Proyecto

Este directorio contiene toda la documentación relacionada con los **registros e informes formales** del proyecto **“Detección de Fibrilación Auricular mediante señales PPG obtenidas de relojes inteligentes”**.

Su propósito es centralizar tanto los **reportes internos del equipo de investigación** como los **informes formales solicitados**.
